# Public share links without public profiles

Dongmodel uses public share links for Share Cards, Group Share Cards, and Wanted Shares so people from external social platforms can view shared items without logging in. We do not provide anonymous public owner profiles in the first version; Owner Galleries are visible only to logged-in Owners and only show Modong that the owner has made gallery-visible. This keeps sharing useful for discovery while avoiding a broader public profile surface before the privacy model is proven.
